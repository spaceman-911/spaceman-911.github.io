# 简介

Axis 是一个开源、基于 XML 的 Web 服务架构。最开始由 IBM 公司研发，叫 IBM-SOAP。后来由 Apache 基金会在 SOAP 的基础上推出了 AXIS。AXIS 本质是一个 Java 语言编写的 SOAP 引擎。利用它我们可以在各种分布式应用中，让各个应用能够彼此操作、交互。
